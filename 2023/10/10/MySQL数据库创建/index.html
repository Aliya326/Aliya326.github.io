MySQL数据库的创建
首页 归档 关于
2023-10-10

创建数据库的基本语法如下：

create database 数据库的名称 if not exists
character set=字符集的名称
collate=collation_name;

说明（以下命令可以根据情况不加入创建数据库的指令，比如创建一个student数据库 create database student）

1.if not exites用于创建前判断该数据库是否存在，当该数据库不存在时才会执行该命令。
此命令可以避免重复创建数据库而造成的错误使后面的命令无法执行。
2.character set=字符集的名称。用来设置该数据库的字符集类型。如果不指定，系统就会使用配置文件中指定的字符集。
3.collate=collation_name。用于指定字符集的排序规则。

另外一些命令

1.show databases命令可以查看当前所有的数据库。
2.show database student可以查看student数据库的信息，如设置的字符集，排序规则。
3.show character可以显示支持的字符集。show collate同理。
