MySQL建表指令如下

use 数据库的名称;
create if not exists table 表名(
列名 数据类型 性质 comment’注释内容’,
……
);

说明

1.性质决定了该列可否为空。主键性质非空且不重复。
2.comment为该列的注释，最多1024个字符

一些性质

1.PRIMARY KEY:定义主键，是一个唯一索引，其中所有主键列必须定义为not null，且不重复，一个表中只能有一个主键。
2.UNIQUE:索引创建一个约束，以便索引中的所有值都必须是不同的。与主键不同的是可以有唯一一个空值，且一个表中可以有多个。
3.NULL或NOT NULL：可以限制字段是否可以为空。
