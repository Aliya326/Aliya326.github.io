指令示例(以book表为例)

1.删除表:delete from book;
2.在表中删除列：alter book drop book_id;
3.在表中增加列:alter book add book_id;
4.更改表中列名:alter book column old_name to new_name;
5.在表中电脑列插入数据:insert into book(book_id,book_name) values(1,’假如给我三天光明’),(2,’数学书’),(3.’英语书’);
6.更改表中列的数据类型:alter table book modify 列名 数据类型 约束;
7.查询指定表或表中的列：select */字段列表 from 表名;
8.更改表中指定列的数据：update book set 列名=数据 where condition;

说明和补充

1.book(book_id,book_name)括号里的内容跟实际表中的列名一一对应。
2.在查询表中 * 意为为查询整张表即select * from boook; 。
另外还有select book_id as name ‘书名’ from book; 意为在查询book_id这个列时给它一个‘书名’的别名。
查询多个列时在后面加一个‘,’即可。
3.限制查询
（一）限制查询行数：select *|列名 from 表名 limit; 。这个命令有两种形式一种是select *|列名 from 表名 limit number; 在number处输入的数字是多少就查几行。
另一种是select *|列名 from 表名 limit number1，number2; 这个指令表示查询表中number1到number2行。
（二）限制查询条件select * from book where condition1 and/or/not condition2; 当指令中的条件为adn时则两个条件都要满足，or则只需其中一个
